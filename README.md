# 🧩 Puzzle Game Pro

A modern and interactive **Sliding Puzzle Game UI** built using **HTML, CSS, and JavaScript**.
This project focuses on clean UI design, smooth animations, and core game logic using minimal technologies.



## 🚀 Features

* 🎮 4x4 Sliding Puzzle Game
* ⏱️ Real-time Timer
* 🔢 Move Counter
* 🏆 Best Score Tracking (Local Storage)
* 🎉 Win Detection System
* 🔄 Shuffle & Restart Options
* ✨ Modern Glassmorphism UI Design
* 📱 Responsive Layout



## 🛠️ Tech Stack

* HTML5
* CSS3 (Grid, Animations, Glassmorphism)
* JavaScript (DOM Manipulation, Game Logic)

## ▶️ How to Run

1. Download or clone this repository:

```
git clone https://github.com/your-username/puzzle-game.git
```

2. Open the project folder

3. Double-click the file:

```
puzzle.html
```

OR open it using a browser.



## 💡 How to Play

* Click tiles adjacent to the empty space
* Arrange numbers from **1 to 15** in order
* Try to finish with **minimum time and moves**
* Your best score will be saved automatically
